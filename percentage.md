## PART 1: Basic Concept (Smart Thinking Approach)

Do not think: "% means divide by 100"  
Think: "% means comparison per 100"

### Must-Memorize Fraction Conversions

| Percentage | Fraction |
|------------|----------|
| 5% | 1/20 |
| 10% | 1/10 |
| 12.5% | 1/8 |
| 16⅔% | 1/6 |
| 20% | 1/5 |
| 25% | 1/4 |
| 33⅓% | 1/3 |
| 40% | 2/5 |
| 50% | 1/2 |
| 66⅔% | 2/3 |
| 75% | 3/4 |
| 80% | 4/5 |

These save 10–15 seconds per question.

## PATTERN 1: x% of y

### Golden Trick

x% of y = y% of x

### Example
25% of 48  
= 48% of 25  
= 12

### Ultra Fast Decimal Shift Trick

40% of 250  
= 4 × 25  
= 100

---

## PATTERN 2: Percentage Increase / Decrease

### Standard Formula

Increase % = (Increase / Original) × 100

### Multiplier Method (Fastest for MCQs)

If increased by 20% → multiply by 1.20  
If decreased by 20% → multiply by 0.80  

Example:
Salary = 5000  
Increase 20%  
New Salary = 5000 × 1.2 = 6000

---

## PATTERN 3: Reverse Percentage (Very Important in TCS)

If final value after 20% increase = 120

Original × 1.2 = 120  
Original = 120 / 1.2 = 100

Always divide by multiplier.

---

## PATTERN 4: Successive Percentage (TCS Favorite Trap)

### Case 1: Two Increases

Net % = a + b + (ab / 100)

Example:  
10% then 20%  
= 10 + 20 + (200/100)  
= 32%

### Case 2: One Increase & One Decrease

Net % = a − b − (ab / 100)

Example:  
+20% then −20%  
= 20 − 20 − 4  
= −4%  

Final Answer = 4% decrease

---

## PATTERN 5: Value Change Questions

A number becomes 120 after 20% increase. Find original.

20% increase → multiply by 1.2

Original × 1.2 = 120  
Original = 120 / 1.2  
= 100

---

## PATTERN 6: Percentage Difference Between Two Numbers

Formula:

Difference / Base × 100

Example:
A = 50  
B = 60  
Difference = 10

B is what % more than A?  
Base = 50  
10/50 × 100 = 20%

A is what % less than B?  
Base = 60  
10/60 × 100 = 16.67%

---

## PATTERN 7: Percentage to Ratio Conversion

If A is 25% more than B  
25% = 1/4  
A : B = 5 : 4

If A is 20% less than B  
20% = 1/5  
A : B = 4 : 5

---

## PATTERN 8: Population Growth

Final = Initial × (1 + r/100)^n

Shortcut for 2 years:

(1 + r/100)^2 = 1 + 2r/100 + (r/100)^2

---

## PATTERN 9: Income – Expenditure – Savings

Assume Income = 100  
Savings = 20 → Expenditure = 80

Income becomes 120  
New Saving = 120 − 80 = 40

Savings doubled → 100% increase

---

## PATTERN 10: Error Based Percentage

Assume number = 100  
Correct 1% = 1  
Wrong 10% = 10  
Difference = 9

% error = (Difference / Correct) × 100

---

## PATTERN 11: Data Interpretation Percentage

Total = 500  
Category A = 150

% = 150 / 500 × 100  
= 30%

## MCQ Elimination Strategy

- If options close → use approximation  
- If option ends with .33 → think 1/3 based  
- If 16.67% appears → base confusion trap  
- If successive % involved → never simple subtraction
